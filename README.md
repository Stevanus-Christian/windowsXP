## How to compile the program code?
- Download and install Node.js [here](https://nodejs.org/en/download/)
- Check the installation with CommandPrompt ‚Üí `node -v`
- Point command prompt to source code storage directory. Example ‚Üí `cd C:/windowsXP-main`
- execute `npm install`
- execute `npm start`
