## How to compile the program code?
- Download and install Node.js [here](https://nodejs.org/en/download/)
- Check the installation with Command Prompt ‚Üí `node -v`
- Point Command Prompt to source code storage directory. Example ‚Üí `cd C:/windowsXP`
- Execute `npm install`
- Execute `npm start`
